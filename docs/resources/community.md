## ğŸ†˜ Getting Help

### Before Asking

1. **Search Documentation**: [docs.neoframework.com](https://docs.neoframework.com)
2. **Search GitHub Issues**: Might already be reported
3. **Search Stack Overflow**: Question might be answered
4. **Check FAQ**: [neoframework.com/faq](https://neoframework.com/faq)

### How to Ask Good Questions

**Include**:
- NeoFramework version
- PHP version
- Operating system
- What you're trying to do
- What you've tried
- Expected vs actual behavior
- Minimal code example
- Error messages

**Example**:
```
I'm trying to create a route with optional parameters but getting a 404.

NeoFramework: 1.5.0
PHP: 8.2
OS: Ubuntu 22.04

Code:
Route::get('/users/{id?}', [UserController::class, 'show']);

Expected: /users and /users/123 both work
Actual: /users/123 works, /users gives 404

Error: None, just 404 page
```

### Where to Ask

**Quick questions**: Discord, Telegram
**Detailed questions**: GitHub Discussions, Stack Overflow
**Bugs**: GitHub Issues
**Feature requests**: GitHub Discussions (Ideas)

### Response Time

**Discord/Telegram**: Minutes to hours
**GitHub Discussions**: Hours to days
**Stack Overflow**: Hours to days
**GitHub Issues**: Days (for official response)
